Radio Buttons

Radio buttons are for when the user must choose a single item out of several options. Choose radios over selects when you want the user to carefully consider the options and need to expose all available options. If there are only two mutually exclusive options, consider using a single checkbox. For example, use a checkbox for “I agree” instead of two radio buttons for “I agree” and “I don’t agree.” Radio buttons use the default browser style. Smaller radio buttons are also available.
